Is Closure FX Builder a fork of Closure Compiler/Templates/Stylesheets?

No. Closure FX Builder is distributed including Google Closure Tools from the official repository and does not use or allow to use other forks.

Why are Google Closure Tools included into Closure FX Builder?

The most important reason is simplifying installation and configuration processes for beginners.

Does Closure FX Builder work with Closure Library?

Yes. Closure FX Builder allows you to compile JavaScript code used Closure Library easily and without additional tools or scripts.

Is Closure FX Builder useful for compiling Closure-based code only?

No. Closure FX Builder uses Closure Compiler and can be used to minify/compile/debug any JavaScript code (pure JavaScript, jQuery, Angular, Backbone etc.).

What are .closure files?

.closure file is a configuration file which contains all needed information about the current build (sources, libraries, options etc.). It allows you to configure build process once, change it when you want and reuse it continuously.
Moreover, you can create and use several .closure configuration files for one application.

What are System Requirements for Closure FX Builder?

Closure FX Builder is a cross-platform application and available for Windows, Unix and Mac OS. Closure FX Builder requires Java 7 or higher

Where to download Java 7?

We recommend to download and use Oracle's distribution of Java 7 from the official Oracle website.

Do I need to download Google Closure Tools separately?

Closure FX Builder already includes Closure Compiler, Closure Templates and Closure Stylesheets.
Nevertheless, if you plan to use Closure Library in your project, you have to download it from Closure Library repository.

Where can I find installation instructions?

Please explore Getting Started page.

Is there a newsgroup or mailing list devoted to Closure FX Builder?

Yes. Please subscribe to DigiArea Email News choosing Closure FX Builder as a theme.

How do I report a bug I have found in Closure FX Builder?

Please use Closure FX Builder Issue Tracker on GitHub.

What information should I include in a bug report?

In addition to the comment describing the bug, you should specify the versions of Closure FX Builder, the Java platform and operating system you are using, because bugs often arise that are specific to particular versions. Sometimes the bug arises from the Java runtime environment or even the operating system, and knowing these details helps the developer team design testing and other debugging strategies.
