Running Closure Stylesheets

In this section, you will learn about running Closure Stylesheets in Closure FX Builder.

  • Running Closure Stylesheets
  • Re-launching and Terminating Closure Stylesheets

Running Closure Stylesheets

  1. Open your Build Configuration and choose Stylesheets page.
  2. On the Stylesheets page, find Run Closure Stylesheets button and press it to run the Stylesheets.

  3. Notice that the Stylesheets has finished running and the following message appears in the Console view telling you that building has finished.

  4. The Stylesheets also reports compilation errors to the console.

  5. Use Problems view to review and manage errors and warnings related to the passed code.

The Run command is also available in other places, such as the main toolbar. You also can use Ctrl+R (Windows, Linux) and Cmd+R (Mac) key binding. In this case, all of the activated Stylesheetss will be launched.

Re-launching and Terminating Closure Stylesheets

Use Run button to launch and relaunch the Stylesheets using current buildpath and options.
Use Cancel button to stop the Stylesheets.
