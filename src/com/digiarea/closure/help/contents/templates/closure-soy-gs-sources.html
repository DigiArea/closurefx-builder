Adding Templates Sources

In this section, you will organize your sources in the Build Path and specify the output location.

  1. Go to Build Path tab on the Templates page. The page contains four sections: Sources, Libraries, Order and Output Location.
  2. Click Add... button in the Source section.
  3. Choose a folder with source files and click OK.

  4. Define Output Path Format. Using Placeholders button you can add a Placeholder. The Compiler uses Output Path Format with Placeholders and generates path for each output file dynamically.

    For example, using the following path format and having Template1.soy and Template2.soy files, the compiler generates Templates1.js and Templates2.js files into the bin folder inside your project.
